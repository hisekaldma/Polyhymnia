About

Polyhymnia is a live programming language for composing music. It is still under heavy design and development. You can try out an early version to the left, or check the documentation for reference.

Instruments

The editor has the following instruments that you can try:

  • Marimba
  • Pad
  • Kick
  • Snare
  • Clap
  • Shake
  • Hihat
